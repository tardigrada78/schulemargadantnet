Verwendbare KI-Modelle

Momentan bietet erst die Abstract-Zusammenfassung eine Wahl des KI-Tools, die Funktion wird aber bald überall implementiert. Untenstehende Varianten stehen zur Auswahl.
Bilder, Text zu Sprache und Sprache zu Text wird unabhängig von OpenAI generiert - die dazu notwendigen Daten werden immer an deren Server übermittelt.

OpenAI Cloud

Die Modelle von OpenAI (z. B. GPT-4o) laufen über die OpenAI-Server; die Daten verlassen Ihr Gerät. Diese API-Calls sind kostenpflichtig.

Anthropic Cloud

Die Modelle von Anthropic (z. B. Claude Sonnet oder Claude Haiku) laufen über die Anthropic-Server; die Daten verlassen Ihr Gerät. Diese API-Calls sind kostenpflichtig.

Ollama via Cloudflare Tunnel Lokal (Installation)

Mit Ollama können KI-Modelle lokal auf dem eigenen Gerät ausgeführt werden – ohne dass Daten an externe Server gesendet werden. Über einen Cloudflare-Tunnel wird das lokal laufende Ollama für die Website erreichbar gemacht. Beides ist gratis.

Anleitung
  1. Cloudflared installieren – Anleitung und Downloads: developers.cloudflare.com → Tunnel Downloads
  2. Ollama installieren und starten – Ollama normal starten (es läuft standardmässig auf Port 11434).
  3. Tunnel öffnen – In der Konsole eingeben:
    cloudflared tunnel --url http://localhost:11434
  4. URL kopieren – Die angezeigte URL (erscheint nach der Meldung „Your quick Tunnel has been created!") in die Zwischenablage kopieren und auf im entsprechenden Feld dieser Website eintragen.
  5. Modell wählen – Auf dieser Website bei «Modell» eines der installierten lokalen Modelle eingeben. Installierte Modelle können mit ollama list aufgelistet werden.

webLLM Lokal (Browser)

webLLM führt KI-Modelle direkt im Browser aus, ohne dass etwas installiert werden muss. Das Modell wird einmalig in den Browser-Cache heruntergeladen und danach vollständig lokal ausgeführt. Voraussetzung ist ein aktueller Browser mit WebGPU-Unterstützung (z. B. Chrome 113+). Je nach Modellgrösse wird mehr Arbeitsspeicher benötigt.
